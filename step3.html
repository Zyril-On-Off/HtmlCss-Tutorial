CSS Reset & Body Styling

Now we start adding CSS! First, we'll reset browser defaults and create a beautiful gradient background.

üìê Why Reset CSS?

Different browsers have different default styles. A CSS reset removes these differences so our page looks the same everywhere!

üé® The CSS Code

Add this inside <style> tags in the <head> section:

/* Reset default browser styles */ * { margin: 0; padding: 0; box-sizing: border-box; } /* Style the body */ body { font-family: 'Arial', sans-serif; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); min-height: 100vh; display: flex; align-items: center; justify-content: center; }

üîç Breaking It Down

The Universal Selector (*)

  • margin: 0 - Removes default spacing outside elements
  • padding: 0 - Removes default spacing inside elements
  • box-sizing: border-box - Makes width/height calculations easier and more predictable

The Body Styling

  • font-family - Sets the default font for all text
  • background: linear-gradient(...) - Creates a beautiful purple gradient background
  • min-height: 100vh - Makes body at least full viewport height (100% of screen)
  • display: flex - Enables flexbox layout
  • align-items: center - Centers content vertically
  • justify-content: center - Centers content horizontally
üí° What's Flexbox?
Flexbox is a CSS layout method that makes it easy to align and distribute items. By adding display: flex to the body, we can easily center the login container both vertically and horizontally!

üåà About the Gradient

The gradient goes from color #667eea (blue-purple) to #764ba2 (deep purple) at a 135-degree angle. This creates a diagonal gradient effect!
