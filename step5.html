Styling Form Groups

Now we'll style the form groups and labels to make them look clean and professional.

üìã The Form Group CSS

.form-group { margin-bottom: 20px; } .form-group label { display: block; margin-bottom: 8px; color: #555; font-weight: 500; font-size: 14px; }

üîç Breaking It Down

Form Group Styling

  • margin-bottom: 20px - Adds space between each form group (Username and Password fields)

Label Styling

  • display: block - Makes the label take up the full width (forces input to next line)
  • margin-bottom: 8px - Adds small space between label and input
  • color: #555 - Medium gray color (softer than black)
  • font-weight: 500 - Makes text slightly bold (500 is medium weight)
  • font-size: 14px - Slightly smaller than default
üí° Why display: block?
By default, labels are inline elements. Setting display: block makes them take up the full width, so the input field will appear on the next line. This creates a clean vertical layout!

üìè Understanding Spacing

Good spacing is key to professional design:

  • 20px between form groups - creates visual separation
  • 8px between label and input - keeps them connected but distinct
