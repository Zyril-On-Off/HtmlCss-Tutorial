Building the Login Form

Now let's add the actual login form! We'll use HTML form elements to create input fields for username and password.

üèóÔ∏è The Form Structure

Here's the code we'll add inside the <body> tag:

<div class="login-container"> <h2>Welcome Back</h2> <form> <div class="form-group"> <label for="username">Username</label> <input type="text" id="username" placeholder="Enter username"> </div> <div class="form-group"> <label for="password">Password</label> <input type="password" id="password" placeholder="Enter password"> </div> <button type="submit">Login</button> </form> </div>

üîë Understanding the Elements

  • <div class="login-container"> - A container to hold everything together
  • <h2> - A heading that says "Welcome Back"
  • <form> - Wraps all form inputs
  • <div class="form-group"> - Groups each label and input together
  • <label for="username"> - Text label for the input. The for attribute connects it to the input's id
  • <input type="text"> - Text input field for username
  • <input type="password"> - Password input (hides the characters you type)
  • <button type="submit"> - Button to submit the form
üí° Important:
The class attributes (like class="login-container") don't do anything yet! They're labels we'll use in CSS later to style these elements. Think of them as name tags.

üìù Input Types

  • type="text" - Regular text input (shows what you type)
  • type="password" - Hides characters for security (shows dots instead)
