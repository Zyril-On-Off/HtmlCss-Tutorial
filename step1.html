HTML Structure Basics

Every HTML page starts with a basic structure. Think of it like the foundation of a house - you need it before you can build anything else!

üìã The HTML5 Document Structure

Let's look at the essential parts:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Login Page</title> </head> <body> <!-- Your content goes here --> </body> </html>

üîç What Does Each Part Do?

  • <!DOCTYPE html> - Tells the browser this is an HTML5 document
  • <html lang="en"> - The root element, contains everything. lang="en" sets language to English
  • <head> - Contains metadata (information about the page)
  • <meta charset="UTF-8"> - Sets character encoding to UTF-8 (supports all languages)
  • viewport meta tag - Makes the page responsive on mobile devices
  • <title> - Text shown in browser tab
  • <body> - Contains all visible content
üí° Important Note:
The HTML structure itself is invisible! It creates the foundation, but you won't see anything displayed yet. In the next step, we'll add visible content.

‚úÖ What You Learned

  • Every HTML page needs a DOCTYPE declaration
  • The <head> contains information ABOUT the page
  • The <body> contains what users SEE
  • Meta tags help with responsive design and character encoding
