Styling the Login Container

Now we'll make the login box beautiful with a white background, rounded corners, and a professional shadow!

üé® The Container CSS

.login-container { background: white; padding: 40px; border-radius: 10px; box-shadow: 0 10px 25px rgba(0, 0, 0, 0.2); width: 100%; max-width: 400px; } .login-container h2 { text-align: center; color: #333; margin-bottom: 30px; font-size: 28px; }

üîç What Each Property Does

  • background: white - Makes the container background white
  • padding: 40px - Adds 40px of space inside the container on all sides
  • border-radius: 10px - Rounds the corners by 10 pixels
  • box-shadow - Creates a shadow below and around the container for depth
  • width: 100% - Container takes full width of its parent
  • max-width: 400px - But never wider than 400px (keeps it nice and compact)
üí° Understanding box-shadow:
The syntax is: horizontal vertical blur color
0 10px 25px rgba(0,0,0,0.2) means:
‚Ä¢ 0 = no horizontal offset
‚Ä¢ 10px = shadow is 10px down
‚Ä¢ 25px = blur radius (makes it soft)
‚Ä¢ rgba(0,0,0,0.2) = black at 20% opacity

üéØ Styling the Heading

We also styled the h2 inside the container:

  • text-align: center - Centers the text
  • color: #333 - Dark gray color (easier on eyes than pure black)
  • margin-bottom: 30px - Adds space below the heading
  • font-size: 28px - Makes the heading larger
