Styling the Login Button

The final piece! Let's create a beautiful, interactive button that users will want to click.

üé® Button CSS

button { width: 100%; padding: 14px; background: linear-gradient(135deg, #667eea 0%, #764ba2 100%); color: white; border: none; border-radius: 6px; font-size: 16px; font-weight: bold; cursor: pointer; transition: transform 0.3s; } /* Lift button when user hovers */ button:hover { transform: translateY(-2px); }

üîç Property Breakdown

  • width: 100% - Button takes full width
  • padding: 14px - Makes button taller (easier to click)
  • background: linear-gradient(...) - Same gradient as the page background
  • color: white - White text color
  • border: none - Removes default button border
  • border-radius: 6px - Rounded corners
  • font-size: 16px - Larger text
  • font-weight: bold - Makes text bold
  • cursor: pointer - Shows hand cursor on hover
  • transition: transform 0.3s - Smooth animation

‚ú® The Hover Effect

The :hover pseudo-class activates when the user moves their mouse over the button:

  • transform: translateY(-2px) - Moves button up 2 pixels
üí° Why cursor: pointer?
By default, buttons show an arrow cursor. cursor: pointer changes it to a hand pointer, which is the universal symbol for "this is clickable!" It improves user experience.

üéØ Transform Property

The transform property lets you move, scale, rotate, or skew elements:

  • translateY(-2px) - Moves element up by 2 pixels (negative Y = up)
  • Combined with transition, this creates a smooth lift effect
